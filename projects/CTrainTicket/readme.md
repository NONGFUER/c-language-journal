显示主菜单
输入1到4之间的数字，选择功能
输入其他数字，返回主菜单

==注册功能
输入用户名，判断是否合法，
不合法则提示错误，返回输入用户名界面
合法则继续输入身份证，再判断身份证是否合法
如果不合法则提示错误，返回输入身份证界面
合法则继续输入第一次密码，再判断密码是否合法，
如果不合法则提示错误，返回输入第一次密码界面
合法则继续输入第二次密码，再判断密码是否一致
不一致则提示错误，返回输入第二次密码界面
如果一致，则注册成功，否则返回注册界面

用户信息保存到文件中

注册用户 将用户信息保存到文件中，但先要读取文件中的用户信息，
然后将新用户信息添加到文件中，再将文件中的用户信息保存到文件中。
