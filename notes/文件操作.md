# 文件操作大纲

本大纲适用于 C 语言文件操作相关知识点整理。

## 1. 文件的基本概念
- 文件与流的区别
- 文件类型（文本文件、二进制文件）

## 2. 文件打开与关闭
- fopen 函数用法
- 文件打开模式（"r", "w", "a", "rb", "wb" 等）
- fclose 函数

## 3. 文件读写操作
- fgetc, fputc 用于字符读写
- fgets, fputs 用于字符串读写
- fread, fwrite 用于二进制读写

## 4. 文件指针与定位
- ftell 获取文件指针位置
- fseek 移动文件指针
- rewind 重置文件指针

## 5. 文件错误处理
- feof 检查文件结尾
- ferror 检查文件错误

## 6. 文件重命名与删除
- remove 删除文件
- rename 重命名文件

## 7. 典型应用场景
- 日志文件写入
- 配置文件读取
- 数据持久化
